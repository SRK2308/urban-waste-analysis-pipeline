# urban-waste-analysis-pipeline
A multi-model AI system for urban waste detection across different real-world scenarios.

## Overview
This project contains 3 models:
- Construction & Demolition Waste Detection  
- Garbage on Road Detection  
- Overflowing Dustbins Detection  

## Features
- Object detection for urban waste categories  
- Trained on real-world datasets  
- Handles varied lighting and cluttered environments  

## Tech Stack
- Python, PyTorch  
- OpenCV, NumPy  
- YOLO-based detection models  

## Output
Detects and classifies waste types for smart city monitoring applications.

Built for CityLens AI Hackathon
